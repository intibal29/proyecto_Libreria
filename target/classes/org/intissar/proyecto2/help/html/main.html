Menú Principal

El menú principal de la aplicación permite gestionar diferentes aspectos de los préstamos bibliotecarios, incluyendo estudiantes, libros, devoluciones, históricos y generación de informes.

Estructura del Menú Principal

El menú principal está organizado de la siguiente manera:

  • Barra de Navegación: Botones para acceder a diferentes vistas de la aplicación.
  • Vista Principal: Muestra los datos correspondientes a la opción seleccionada.
  • Opciones de Ayuda: Permite consultar guías y cambiar configuraciones, como el idioma.

Funcionalidades Disponibles

Opciones del Menú

  • Gestión de Alumnos: Permite visualizar, añadir, editar o eliminar información de estudiantes registrados.
  • Gestión de Libros: Ofrece herramientas para administrar los libros disponibles en la biblioteca.
  • Gestión de Préstamos: Registra nuevos préstamos, marca devoluciones y consulta préstamos activos.
  • Gestión de Devoluciones: Marca los préstamos como devueltos, actualizando el estado del libro.
  • Histórico de Préstamos: Muestra un listado completo de los préstamos ya devueltos.
  • Generación de Informes: Permite generar informes detallados en diferentes formatos.

Barra de Navegación

La barra de navegación contiene botones para acceder rápidamente a las vistas principales de la aplicación:

  1. Alumnos: Redirige a la vista de gestión de estudiantes.
  2. Libros: Redirige a la vista de gestión de libros.
  3. Préstamos: Abre la vista para registrar y consultar préstamos.
  4. Devoluciones: Permite procesar devoluciones de libros prestados.
  5. Histórico: Accede a la vista del historial de préstamos.

Consejos de Uso

  • Utiliza los botones de navegación para cambiar entre vistas rápidamente.
  • Consulta la sección de ayuda para obtener más detalles sobre cómo usar cada funcionalidad.
  • Los datos en cada vista se actualizan automáticamente tras realizar operaciones como añadir o eliminar registros.

Notas Técnicas

  • Los datos mostrados en cada vista provienen de la base de datos, y cualquier cambio se sincroniza automáticamente.
  • La aplicación utiliza atajos de teclado para acceder a funcionalidades comunes (por ejemplo, Ctrl+N para añadir un registro).
  • El idioma de la interfaz puede cambiarse desde la opción de Ayuda en el menú principal.

Ayuda Adicional

Si tienes dudas o encuentras algún problema, consulta al administrador del sistema o revisa la documentación oficial de la biblioteca.
