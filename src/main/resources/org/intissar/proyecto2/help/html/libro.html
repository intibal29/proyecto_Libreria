Gestión de Libros

La funcionalidad de "Gestión de Libros" te permite administrar los libros disponibles en la biblioteca. Puedes añadir nuevos libros, modificarlos, marcarlos como dados de baja y visualizarlos en una tabla.

Sección de Detalles del Libro

  • Título: Campo para ingresar el título del libro.
  • Autor: Campo para ingresar el nombre del autor del libro.
  • Editorial: Campo para ingresar la editorial que publicó el libro.
  • Estado: Menú desplegable con las opciones:
    • Nuevo
    • Usado nuevo
    • Usado seminuevo
    • Usado estropeado
    • Restaurado

Sección de Tabla

La tabla de libros muestra la información de los libros registrados en el sistema. Incluye las siguientes columnas:

  • Código: Identificador único del libro (no editable).
  • Título: Título del libro.
  • Autor: Nombre del autor.
  • Estado: Estado actual del libro.

Puedes seleccionar un libro de la tabla para modificarlo o marcarlo como dado de baja.

Botones de Control

  • Agregar Libro: Añade un nuevo libro a la base de datos.
  • Modificar Libro: Actualiza la información del libro seleccionado.
  • Marcar como Baja: Marca el libro seleccionado como "dado de baja", excluyéndolo del catálogo activo.

Pasos para Gestionar Libros

  1. Completa los campos de "Título", "Autor" y "Editorial".
  2. Selecciona el estado del libro en el menú desplegable.
  3. Haz clic en "Agregar Libro" para guardarlo en la base de datos.
  4. Para modificar un libro, selecciónalo en la tabla, realiza los cambios y haz clic en "Modificar Libro".
  5. Para marcar un libro como dado de baja, selecciónalo en la tabla y haz clic en "Marcar como Baja".

Consejos de Uso

  • Todos los campos son obligatorios para registrar o modificar un libro.
  • Los libros dados de baja ya no aparecerán en el catálogo activo, pero su información permanecerá en el sistema.
  • Si necesitas cancelar una operación, simplemente no guardes los cambios y selecciona otro libro o limpia los campos.

Notas Adicionales

  • Los datos de los libros se sincronizan automáticamente con la base de datos al realizar cambios.
  • La tabla de libros se actualiza automáticamente después de agregar, modificar o marcar un libro como dado de baja.

Ayuda Adicional
